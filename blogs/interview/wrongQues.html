    # 高级前端面试小程序错题

    1、输出什么

    // index.js
console.log('running index.js');
import {sum} from "./sum"
console.log(sum(1,2));

// sum.js
console.log("running sum.js");
export const sum = (a,b)=>a+b
    1
    2
    3
    4
    5
    6
    7
    8

    答案：running sum.js,running index.js,3

    解析：import 命令是在编译阶段执行的，在代码运行之前。因此这意味着被导入的模块会先运行，而导入模块文件会后执行。这是 CommonJS 中 require（）和 import 之间的区别。require（）在执行阶段运行。

    2、不能像使用常规对象那样向构造函数添加属性。如果要一次向所有对象添加功能，则必须使用原型。

    3、一般情况下当我们将对象赋值为 null,那些对象会被进行垃圾回收，因为已经没有对这些对象的引用了。即：只要存在引用，对象就不会被垃圾回收。

    # 2、meta 标签是干什么的，都有什么属性和作用

    参考答案 元信息

    字符编码。charset="UTF-8"

    页面视口设置，控制页面在移动设备上的显示和缩放行为。name="viewport" content="width=device-width, initial-scale=1.0"

    搜索引擎优化（SEO）。提供描述性信息，便于搜索引擎索引。name="description|keywords" content="页面描述信息"

    页面作者。name="author" content="作者姓名"

    # 3、什么是 DOM ，它和 HTML 有什么区别？

    参考答案

    document object model，js对象，树形结构，可编程，动态性；

    hypertext markup language，超文本标记语言

    HTML 到 DOM 的过程，dom + cssom = render tree，根据渲染树将内容显示在页面上

    # 4、DOM 节点的 attr 和 property 有何区别

    attr 指的是 HTML 属性（attribute），property 指的是 DOM 对象的属性（property）

    主要区别

    定义不同

    存储位置不同

    行为不同

    总结，一般来说，attr 用于设置元素的初始状态，而 property 用于操作和获取当前状态。
